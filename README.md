# ABP - PROGRAMACIÓN

Este repositorio contiene el código fuente y la estructura de la base de datos, desarrollado como parte de la metodología de Aprendizaje Basado en Proyectos (ABP) en el ámbito de la programación. El sistema está orientado a la gestión o simulación de redes de fibra óptica.

## 🚀 Estructura del Proyecto

El repositorio está compuesto por los siguientes archivos principales:

*   **`FIBRACORD.PY`**: Módulo principal en Python que contiene la lógica de negocio para la gestión de conexiones, nodos o cableado de fibra.
*   **`SCIPT.PY`**: Script ejecutable de Python encargado de la automatización, pruebas de ejecución o la interfaz de comandos con el usuario.
*   **`base-fibra.sql`**: Script SQL que contiene el diseño de la base de datos (tablas, relaciones y datos iniciales) para el almacenamiento de la información del sistema.

## 🛠️ Tecnologías Utilizadas

*   **Python 3.x** (100% del código de la aplicación)
*   **MySQL** (Estructura de datos en `base-fibra.sql`)

## 📦 Instalación y Configuración

### 1. Clonar el repositorio
```bash
git clone https://github.com/serrasofia1998-dev/ABP-PROGRAMACION.git
cd ABP-PROGRAMACION
```

### 2. Configurar la Base de Datos
1. Abra su gestor de bases de datos preferido.
2. Importe y ejecute el archivo `base-fibra.sql` para crear las tablas necesarias.

### 3. Ejecutar la Aplicación
Asegúrese de tener Python instalado y ejecute el script principal:
```bash
python SCIPT.PY
```
